Monte Carlo Simulation to Estimate Portfolio Value-at-Risk

By Gloria and team | STAT0011 Project

üìà Overview

This project aims to assess the market risk of a portfolio comprising two major financial indices: the DAX and the CAC 40. Using weekly price data from 2000 to 2023, we estimate the portfolio‚Äôs 99% and 95% Value-at-Risk (VaR) through a Monte Carlo simulation based on copula theory.

üî¨ Methodology

The analysis followed a structured modelling process:

  • Calculated log returns for both indices and analysed autocorrelation
  • Fitted AR + GARCH models to capture volatility clustering
  • Used information criteria (AIC/BIC) to choose optimal conditional distributions
  • Confirmed model adequacy with residual diagnostics and statistical tests
  • Applied copula modelling (BB1) to simulate joint dependencies between assets
  • Performed Monte Carlo simulations to estimate portfolio VaR

üß™ Tools & Libraries

  • R programming (GARCH, Copula, VineCopula packages)
  • Monte Carlo simulation (10,000 paths)
  • Kolmogorov-Smirnov & Anderson-Darling tests for model validation

üìä Results

The simulation estimated the portfolio's Value-at-Risk as follows:

  • 99% VaR: -0.00777
  • 95% VaR: -0.00475
